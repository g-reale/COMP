◆ scan()

void scan ( unsigned char  c)

Scan a single character from the input. This scanner is a switch case implementation of a dfa. It will consume chars trying to detect transitions. When a transition is detected categorize is called. Each transition/rule has the format:

<open><sources><sep><destination><sep><transitions><close>
