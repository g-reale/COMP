Detailed Description

DFA based scanner. This file defines the prototypes for the scanner abstraction. The scanner must be able to tokenize a program stored in a buffer via the getToken call. Currently the scanner uses a dfa table for categorizing the lexemes where the ambiguity between keywords and identifiers is resolved using a hash.
